Dependencies:

This code requires the pcg-cpp headers to compile [https://www.pcg-random.org/]

  pcg_extras.hpp
  pcg_random.hpp

They are available at https://github.com/imneme/pcg-cpp/tree/master/include
and need to be placed in the same directory as this README.

Compilation:

$ make

Tested with g++ 12.2.0
